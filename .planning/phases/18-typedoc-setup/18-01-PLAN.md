  <action>
Install TypeDoc and create the documentation generation setup:

1. **Install typedoc** as a dev dependency:
   ```bash
   npm install --save-dev typedoc@^0.28.17
   ```
   Note: Version 0.28.9+ adds TypeScript 5.9 support (resolves STATE.md blocker).

2. **Create typedoc.json** at project root with:
   ```json
   {
     "$schema": "https://typedoc.org/schema.json",
     "entryPoints": ["src/opencarl/"],
     "out": "docs",
     "readme": "./README.md",
     "gitRemote": "origin",
     "gitRevision": "main",
     "sourceLinkExternal": true,
     "categorizeByGroup": false,
     "defaultCategory": "Other",
     "categoryOrder": ["Configuration", "Loading", "Matching", "Injection", "Signals", "Errors", "*"],
     "navigation": {
       "includeCategories": true
     },
     "excludePrivate": true,
     "excludeProtected": false,
     "excludeInternal": true,
     "hideGenerator": false,
     "githubPages": true
   }
   ```

   Key configuration decisions (per CONTEXT.md locked decisions):
   - `entryPoints: ["src/opencarl/"]` - documents all TypeScript in src/opencarl/
   - `readme: "./README.md"` - README as documentation homepage
   - `gitRemote: "origin"` - links to KrisGray/opencarl (not upstream)
   - `gitRevision: "main"` - source links to main branch
   - `sourceLinkExternal: true` - opens links in new tab
   - `categoryOrder` - predefined categories for Phase 18-02 implementation
   - Default theme with built-in dark/light toggle (no custom styling)

3. **Add docs script** to package.json scripts:
   ```json
   "docs": "typedoc",
   "docs:ci": "typedoc --gitRevision $(git rev-parse HEAD)"
   ```
   The `docs:ci` script is for Phase 19 CI deployment with exact commit hash.

4. **Add docs/ to .gitignore** (if not already present):
   ```
   # Generated documentation
   docs/
   ```
   Docs are generated locally or deployed to gh-pages (Phase 19), not committed to main.

DO NOT:
- Add custom CSS/theming (default theme per user decision)
- Configure deployment (Phase 19 scope)
- Add @category tags to source files (Phase 18-02 task)
</action>
